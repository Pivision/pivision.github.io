寻找整理我心目中的 Windows 佳软

软件设计有两种方式：一种方式是，使软件过于简单，明显没有缺陷；另一种方式是，使软件过于复杂，没有明显的缺陷。——C.A.R. Hoare

安全类

  • 火绒

    简介：强悍、 轻巧、 干净，没有任何广告，从来不会去打扰你，以致于你可能忘记安装过它。

    地址：https://www.huorong.cn

  • Windows Defender

    简介：系统自带，满足日常需要，但是效果太强被我禁用掉。

    地址：Windows 10 自带

输入法

  • 微软输入法

    简介：Windows 10 自带输入法，简洁美观，日常够用。

    地址：Windows 10 自带

  • 手心输入法

    简介：无广告，无骚扰，简约美观，还有多款皮肤，是款良心产品，就是更新周期有点长。

    地址：http://www.xinshuru.com

浏览器

  • 百分浏览器

    简介：基于 Chromium 引擎系统，继承了所有 Chrome 浏览器的优点，又进行了适度的增强。

    地址：https://www.centbrowser.cn

  • Edge 浏览器

    简介：同样 Windows 10 自带，日常够用，目前作为备用。

    地址：Windows 10 自带

办公类

  • Office Tool Plus

    简介：一个小巧、安全、简洁的 Office 管理、下载、安装器。

    地址：https://otp.landian.la

下载类

  • Internet Download Manager

    简介：Internet Download Manager 的大名自然不用多说，十分强大。

    地址：https://www.internetdownloadmanager.com

  • PanDownload

    简介：破解百度云限速，良心软件。

    地址：https://pandownload.com

    凉了

通讯类

  • QQ

    简介：相信不用介绍，每个人都会用到，有许多第三方修改版很好用。

    地址：https://im.qq.com

  • 微信 UWP

    简介：UWP 应用还是很好用的，比其他版本省心的多。

    地址：Microsoft Store 搜索

系统类

  • Dism++

    简介：也许是最强的实用工具，包括空间回收、系统优化、更新管理等许多功能，不可多得的绿色实用软件。

    地址：https://www.chuyu.me

  • Bandizip

    简介：免费优秀的文件压缩/解压缩软件，无广告。

    地址：https://www.bandisoft.com/bandizip

音乐类

  • 网易云音乐 UWP

    简介：简洁好用，可惜没有桌面歌词。

    地址：Microsoft Store 搜索

视频类

  • PotPlayer

    简介：老牌子，轻巧、强大、精美，本地播放器首选。

    地址：http://potplayer.daum.net

  • 哔哩哔哩 UWP

    简介：强力安利，大神开发的第三方，美观强大。

    地址：Microsoft Store 搜索

清理类

  • CCleaner

    简介：清理 Windows 个人电脑的顶级工具，免费、轻巧、快速、功能强大、清理彻底。

    地址：https://www.ccleaner.com/

  • GeekUninstaller

    简介：一个免费的为“极客”们量身定做的专业级单文件绿色卸载工具，可以保证卸载无残留。

    地址：https://geekuninstaller.com/

美化类

  • 雨滴桌面

    简介：很多人都在用的桌面美化工具，小巧，强大。

    地址：https://www.rainmeter.net

  • TranslucentTB

    简介：能将任务栏透明化，支持多种样式。

    地址：Microsoft Store 搜索

  • Tile Genie

    简介：为你增加想要的图片磁贴。

    地址：Microsoft Store 搜索

推荐殁漂遥，一个不以营利为目的的分享网站，第一时间免费分享实用的软件、电影、系统等。坚持优化，绿色。
