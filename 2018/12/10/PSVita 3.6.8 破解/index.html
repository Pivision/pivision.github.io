等了将近一个月的 PSV 终于到货，一到手便寻找资料将其破解，再次整理一下破解过程。

顺便期待一波 3.69 的完美破解。

准备阶段

  • 电脑安装内容管理助手
  • 已安装驱动程序
  • 已下载 Final h-encore
  • PSV 已登陆 PSN 账号

密码：cgpo

破解阶段

  1. 打开 QcmaDriver_winusb.exe 安装驱动

  2. 解压 FinaIHE_v1.x_win32_full.7z

  3. 打开 FinaIHE.exe

  4. 打开内容管理助手

  5. 连接 PSV 与电脑

  6. 点击开始并等待进度条结束

  7. 操作 PSV 选择

    • 电脑 —> PS Vita

      应用程序

      PS Vita

    • 里面有个图标是锁头的，左边有个可以勾选的，勾选它。然后点击右下角的“复制”。
      会出现进度条，等他进度条走完以后
      返回桌面

    • 打开PSV里面安装好的软件
      打开以后是这样子的
      [*] Exit
      [ ] Install HENkaku
      [ ] DownIoad VitaShell
      [ ] Reset taiHEN config.txt

    • 依次选择 2、3、4、1
      进入设置看如果是 3.68 变革
      即破解完成

使用阶段

  • 安装 vita工具箱

  • 安装 PKGJ

  • 安装 vpk 方法

    进入 VitaShell，按 select 按键，将 PSV 与电脑连接

    将 vpk 文件拖到根目录，安装后可删除
