近日发现博客登不上去，有时淘宝和少数派也上不去，换了手机热点才发现是校园网的问题 = =///

查阅相关资料后发现修改默认 DNS 地址便可以解决，在此备份一下，以备后用。

P.S. 本文环境为 Windows 10 系统

准备

在列表教育网 DNS 服务器列表&国内各地区主要的 DNS 服务器列表中找到与你网络接入商所在地最近的两个 DNS 服务器 ip

修改 DNS 地址

  1. 进入设置——网络和 Internet

  2. 点击更改适配器选项

  3. 选中你所用的网络连接，右键点击属性

  4. 双击 Internet 协议版本 4

  5. 将选择好的 DNS 服务器填入，保存，重新连接网络即可
